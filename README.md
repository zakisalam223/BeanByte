# BeanByte

A web application created as a forum for computer science students to collaborate and share knowledge.

## Features:

- Users can register and edit their account details
- Users can create thread sections for various topics
- Users can create threads and reply to others within thread sections
- Communicate directly with other users through private messages
- Share files within threads or private messages

## Built with:
- ASP.NET Core  
- MongoDB

Created in 2024
